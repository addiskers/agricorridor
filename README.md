# AgriCorridor Global

South-South Agricultural Technology Transfer Platform — connecting India and Ethiopia through agri-tech innovation.

## Features

- Interactive dashboard with regional agricultural data
- AgroMatch technology matchmaking system
- Marketplace for agricultural products and services
- AI-Adapt climate adaptation tools
- Farmer mobile app (USSD) mockup
- Mentor network directory
- Finance and regulatory compliance tools
- HS code / duty calculator and document checklist wizard

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Lora + DM Sans)
- Fully static — no server required

## Deployment

This project is deployed as a **Static Site** on [Render](https://render.com).

### Deploy to Render

1. Push this repo to GitHub
2. Connect the repo on Render as a **Static Site**
3. Set **Publish Directory** to `public`
4. Deploy

Or click the button below:

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)

## Local Development

Simply open `public/index.html` in any modern browser.
